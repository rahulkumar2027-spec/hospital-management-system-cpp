# Hospital Management System

A console-based Hospital Management System built in C++ to demonstrate core
Object-Oriented Programming concepts — inheritance, multiple and virtual
inheritance, polymorphism, encapsulation, operator/function overloading,
friend functions, exception handling, and file handling.

The full project, build instructions, and a concept-by-concept breakdown are
in [`Hospital/README.md`](Hospital/README.md).

## Quick Start

```bash
cd Hospital
g++ -Wall -Wextra -std=c++17 Code.cpp -o hospital
./hospital
```
